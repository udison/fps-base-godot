# Godot FPS Base

A base for FPS/Immersive sim games made in Godot 4.3.

## Features

- State machine based player controller
- Walk
- Jump
- Leaning
- Crouch
- Prone
- Debug overlay
- Crosshair
- Interaction system

## Roadmap

- [ ] Raytrace weapons
- [ ] Projectile weapons
- [ ] Enemies


## Usage

Clone this repository with
```
git clone https://github.com/udison/fps-base-godot.git
```
and happy coding!
